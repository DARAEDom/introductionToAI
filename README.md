# Sudoku AI Solver

This python program is capable of finding solutions to 9x9 Sudoku Boards using Backtracking technique. 
Graphical Interface is based on TkInter Libraries for Python.

## Run the application
Before running make sure you are using Python3 3.4 or later version
There are two ways of starting this program, you can either open GUI.py which will display empty board or solver.py that will display solved sudoku board.
On Windows you can open it using IDE or Python IDLE
On Linux you can use IDE or Linux terminal typing ``python3 solver.py`` or ``python3 gui.py``
